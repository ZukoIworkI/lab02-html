# Практична робота: Основи CSS. Оформлення багатосторінкового сайту

Цей проєкт є виконанням практичної роботи №2, присвяченої закріпленню навичок роботи з CSS, створенню адаптивної верстки та використанню Flexbox.

## 🎯 Мета роботи
Закріпити навички підключення зовнішніх стилів, налаштування типографіки, кольорової палітри та побудови адаптивного інтерфейсу для багатосторінкового сайту.

```text
